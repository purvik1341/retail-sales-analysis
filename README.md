# Retail Sales & Profit Analysis Dashboard

## Overview

This project analyses retail sales data to uncover insights on sales trends, regional performance, and product category profitability using SQL and Power BI.

## Objectives

* Identify sales trends over time
* Compare performance across regions
* Analyse profit contribution by category

## Tools Used

* SQL (Data cleaning and aggregation)
* Excel (Data pre-processing)
* Power BI (Dashboard creation and visualization)

## Dataset

Urban Cart Superstore dataset (cleaned and processed)

## Key Insights

* West region generated the highest sales
* Office Supplies category contributed the highest profit
* Sales show an overall increasing trend over time

## Files Included

* Power BI Dashboard (.pbix)
* Cleaned datasets (.csv)
* Project report (.pdf)
